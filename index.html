HOW TOUSE

      
Step-by-step guide

    


    
1
Copy YouTube URL
유튜브 영상 주소를 복사하세요.
예: https://youtu.be/XXXXXXX

    
2
EXTRACT 탭 → Paste & Generate
PASTE 버튼으로 URL 붙여넣기 후
GENERATE COMMAND 클릭

    
3
Terminal에서 실행
생성된 커맨드를 복사해
PC 터미널에서 실행하면 음원 파일이 저장됩니다.
yt-dlp와 ffmpeg 설치 필요

    
4
PLAYER 탭 → 재생
저장된 음원 파일을 불러와
반복·셔플·재생목록 기능으로 감상

    
5
EDITOR 탭 → 편집
파일을 불러와 구간 자르기, 페이드인/아웃 효과 설정 후
ffmpeg 커맨드를 생성해 PC에서 실행


    
💡 지원 형식MP3 · WAV · FLAC · M4A · AAC · OPUS

브라우저 설정에 따라 재생이 유지될 수 있습니다.
