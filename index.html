HOW TO USE

      
Step-by-step guide

    

    
1
Copy YouTube URL
유튜브 주소 복사
예: https://youtu.be/XXXXXXX

    
2
EXTRACT → Paste & Generate
PASTE 후 GENERATE COMMAND 클릭

    
3
Terminal에서 실행
커맨드를 복사해 PC 터미널에서 실행
yt-dlp와 ffmpeg 설치 필요

    
4
PLAYER → 재생
파일 불러와 반복·셔플·재생목록으로 감상

    
5
EDITOR → 편집
파형 드래그로 구간 선택, START/END 슬라이더로 정밀 조정
페이드 효과 후 ffmpeg 커맨드 생성
