HOW TO USE

      
Step-by-step guide

    

    
1
Copy YouTube URL
유튜브 영상 주소를 복사하세요.
예: https://youtu.be/XXXXXXX

    
2
EXTRACT → Paste & Generate
PASTE 버튼으로 URL 붙여넣기 후
GENERATE COMMAND 클릭

    
3
Terminal에서 실행
생성된 커맨드를 PC 터미널에서 실행
yt-dlp와 ffmpeg 설치 필요

    
4
PLAYER → 재생
음원 파일을 불러와 반복·셔플·재생목록으로 감상

    
5
EDITOR → 편집
파형 드래그로 구간 선택, 줌인/아웃으로 정밀 편집
페이드인/아웃 효과 적용 후 ffmpeg 커맨드 생성
