HOW TOUSE

Step-by-step guide to extract audio from YouTube

1
Copy YouTube URL
YouTube에서 원하는 영상을 열고
주소창의 URL을 복사하세요.
예: https://youtu.be/XXXXXXX
2
Paste & Set Options
EXTRACTOR 탭으로 이동 후
PASTE 버튼을 눌러 URL을 붙여넣고
원하는 형식(MP3 등)과 품질을 선택하세요.
3
Generate Command
GENERATE COMMAND 버튼을 누르면
터미널에서 실행할 커맨드가 자동 생성됩니다.
4
Install Dependencies
처음 사용 시 아래 패키지를 먼저 설치하세요.
yt-dlp — 유튜브 다운로더
ffmpeg — 오디오 변환 엔진
5
Run in Terminal
생성된 커맨드를 COPY 후
PC의 터미널(macOS: Terminal, Windows: CMD 또는 PowerShell)에 붙여넣고 실행하면
현재 폴더에 음원 파일이 저장됩니다.
💡 지원 형식 안내 MP3 — 가장 범용적, 스트리밍 추천
WAV — 무손실, 용량 큼
FLAC — 무손실 압축, 음질 최우선
M4A / AAC / OPUS — 모바일 최적화
